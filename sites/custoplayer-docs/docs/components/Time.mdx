# Time

There are 2 time components
1. `"currentTime"`
2. `"duration"`

## ⌛️ Current Time
The currentTime component shows the current time of the video.

### Usage
```js
values={{
  item1: {
    id: 'currentTime',
    textColor: '#b7cef4',
  },
}}
```

* Changes the color of the time text.
  * Accepts any hex or rgb color code.


## ⏳ Duration
The duration component shows the duration of the video.

### Usage
```js
values={{
  item1: {
    id: 'duration',
    textColor: '#b7cef4',
  },
}}
```

* Changes the color of the time text.
* Accepts any hex or rgb color code.
