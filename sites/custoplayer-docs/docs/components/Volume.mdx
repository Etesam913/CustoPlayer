## Usage
```jsx
item1: {
  id: 'volumeButton1',
  barId: 'volumeBar2',
  volumeColor: '#a4c3f5',
  barColor: "white",
  buttonColor: "#a4c3f5",
  scrubberColor: "#a4c3f5",
  scrubberBorderColor: "white"
},
```

* The `barId` property accepts values of "volumeBar1" or "volumeBar2".
  * volumeBar1 is horizontal and volumeBar2 is vertical.
* The `volumeColor` property changes the color of the volume progress in the bar.
  * Accepts any hex or rgb color code.
* The `barColor` property changes the volume background color of the bar.
  * Accepts any hex or rgb color code.
* The `buttonColor` property changes the color of the volume button icon.
  * Accepts any hex or rgb color code.

* The `scrubberColor` property changes the color of the scrubber (the circle at the end of the progress).
  * Accepts any hex or rgb color code
  * If no `scrubberColor` is defined, the `scrubberColor` is defaulted to the same as the `progressColor`


* The `scrubberBorder` property changes the border color of the scrubber (the circle at the end of the volume progress).
  * Accepts any hex or rgb color code
  * If no `scrubberBorderColor` is defined, the `scrubberBorderColor` is defaulted to a lightened version of the `scrubberColor`

* The `hideOnMobile` property hides the volumeButton & volumeBar component when the video's width is less than 768px.
  * Accepts values of `true` or `false`
