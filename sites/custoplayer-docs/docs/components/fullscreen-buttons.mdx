## Usage

```js
item1: {
  id: 'fullscreenButton1',
  buttonColor: '#b7cef4',
},
```

- The `buttonColor` property changes the color of the fullscreen button.
  - Accepts any hex or rgb color code.
- The `hideOnMobile` property hides the fullscreen button when the video's width is less than 768px.
  - Accepts values of true or false
