# Play Buttons

There are currently two playButtons

1. `"playButton1"`
2. `"playButton2"`

## Usage

```jsx
item1: {
  id: 'playButton1',
  buttonColor: '#a4c3f5',
},
```

- The `buttonColor` property changes the color of the playButton.

  - Accepts any hex or rgb color code.

- The `hideOnMobile` property hides the playButton component when the video's width is less than 768px.
  - Accepts values of `true` or `false`
